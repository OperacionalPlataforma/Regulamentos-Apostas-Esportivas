Quais são as regras para uso?

Para cumprimento do rollover/requisitos, é preciso apostar nos jogos das principais ligas.

Serão utilizados os fundos de depósitos para cumprimento do rollover. Se, por ventura, estes fundos sejam utilizados e não haja mais nenhum valor disponível, o saldo de bônus será utilizado para o cumprimento do rollover.

Entretanto, para palpites com valores acima do valor real disponível, serão utilizados os valores de bônus para complementar o valor desejado na aposta. Por exemplo: possuo R$50 de fundo real e R$50 de saldo de bônus. Desejo realizar uma aposta de R$75. Será utilizado os R$50 do fundo real mais R$25 do saldo de bônus. Se a aposta for vencedora, os ganhos serão distribuídos de acordo com a proporção apostada entre o fundo real e o saldo do bônus.

O bônus é válido apenas para apostas pré-live.

O rollover somente será contabilizado após a aposta ser liquidada. Apostas em aberto não contabilizarão para o rollover.

Apostas realizadas em duplicidade, ou seja, apostas realizadas mais de uma vez no mesmo jogo, somente contarão para o rollover a primeira aposta. As apostas seguintes no mesmo evento não serão contabilizadas.

Lances feitos com fundos de bônus, serão automaticamente adicionados ao saldo de bônus dos jogadores e estarão disponíveis para retirada após o cumprimento do rollover.

O lance máximo aceito para cumprimento do rollover/requisitos é de R$100,00. Em lances com valores maiores, não contam para o cumprimento do rollover.

Para bilhetes simples, é preciso que a odd seja igual ou superior a 2.0.

No contexto de apostas múltiplas, é imperativo que todos os eventos incluídos no bilhete atendam integralmente aos requisitos estipulados. Cada evento promocional no bilhete deve apresentar odds mínimas de 1.50 para garantir a conformidade com as condições estabelecidas.

Esta oferta não pode ser combinada com nenhuma outra oferta.

Para cumprimento do rollover/requisitos, somente são aceitos jogos de futebol das ligas citadas acima e NBA, excluindo esportes virtuais, cassinos e e-sports.

Jogos nos quais são feitos cashout, não contarão para o cumprimento do rollover.

O rollover não é válido para pagamento antecipado.

Os requisitos/rollover precisam ser cumpridos no prazo de 7 dias a partir do registro. Após 7 dias, o bônus e todos os ganhos obtidos por ele serão expirados.

Quais são as regras para realizar o saque do saldo de bônus?

O saldo de bônus pode ser retirado a qualquer momento, desde que o jogador cumpra o rollover estabelecido pela casa.

O jogador deve movimentar pelo menos 15X o valor depositado mais o valor de bônus para conseguir efetuar o saque (rollover).

O cliente concorda que o ganho máximo após completar o rollover com o saldo de Bônus é ( mil reais).

Ex: depositando R$50, você ganha R$50 de bônus. Suponha que você aposte R$25 e perca, você ficará com R$75 de saldo (R$25 do depósito e R$50 de bônus). Caso opte pelo saque desse valor, além do saldo de bônus restante ser retirado. Logo, você permanecerá com R$25.

Usuários participantes estão sujeitos a análise antifraude antes da retirada.
